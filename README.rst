StraWBerryPy
============

StraWBerryPy (Single-poinT and local invaRiAnts for Wannier Berriologies in Python) is a Python package for reading tight-binding models from `PythTB <http://www.physics.rutgers.edu/pythtb/>`_, `TBmodels <https://tbmodels.greschd.ch/en/latest/>`_ and `Wannier90 <https://wannier.org/>`_ (through `WannierBerri <https://wannier-berri.org/index.html>`_). Using those models supercells and finite models can be created, and their topological invariants (single-point or local marker) can be computed.
